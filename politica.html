O que o PinFlow faz

Descrição oficial da finalidade e uso das permissões do aplicativo PinFlow junto à API do Pinterest.

🎯 Descrição do Aplicativo

O PinFlow é uma ferramenta de automação de marketing para o Pinterest. Seu objetivo é permitir que criadores de conteúdo, blogueiros e empresas gerenciem e publiquem Pins de forma eficiente, seja manualmente, de forma agendada ou automaticamente a partir de fontes RSS.

O aplicativo é destinado exclusivamente ao uso pelo próprio titular da conta Pinterest autenticada, nunca atuando em nome de terceiros sem consentimento.

⚙️ Funcionalidades Principais

  • Criação de Pins: o usuário cria Pins com imagem, título, descrição e link de destino diretamente pelo PinFlow.
  • Agendamento: os Pins podem ser agendados para publicação em data e hora específicas, de forma automática.
  • Publicação em lote: suporte ao upload de até 500 imagens por vez, gerando múltiplos Pins com um único fluxo de trabalho.
  • Gerador de conteúdo com IA: uso de inteligência artificial para sugerir títulos, descrições e hashtags otimizadas para SEO no Pinterest.
  • Automação via RSS: monitoramento de feeds RSS de blogs e sites para publicar automaticamente novos conteúdos no Pinterest.
  • Gerenciamento de Boards: leitura e seleção dos quadros (boards) da conta autenticada para organizar os Pins publicados.

🔐 Permissões Solicitadas à API do Pinterest

O PinFlow solicita apenas as permissões estritamente necessárias para seu funcionamento:

Permissão (Scope) Para que é usada Necessária
pins:write Criar e publicar novos Pins na conta do usuário ✓ Sim
pins:read Listar Pins existentes para evitar duplicatas ✓ Sim
boards:read Listar os quadros da conta para o usuário escolher onde publicar ✓ Sim
boards:write Não solicitado ✗ Não
user_accounts:read Exibir nome e foto da conta conectada na interface ✓ Sim

👤 Público-alvo

  • Criadores de conteúdo que desejam automatizar a presença no Pinterest
  • Blogueiros que querem distribuir posts automaticamente via RSS
  • Pequenas e médias empresas que gerenciam marketing visual
  • Profissionais de marketing digital que gerenciam múltiplos Pins
✅ Uso exclusivo pelo dono da conta
O PinFlow nunca publica conteúdo sem ação direta ou configuração prévia do usuário autenticado. Não há acesso a contas de terceiros.
🚫 O que o PinFlow NÃO faz
O app não vende, compartilha ou monetiza dados da conta Pinterest. Não realiza ações automatizadas além das configuradas explicitamente pelo usuário. Não acessa mensagens privadas nem dados financeiros.
🔒 Política de Privacidade

Como tratamos seus dados

Esta política descreve quais informações o PinFlow coleta, como as utiliza e como as protege.

📄 1. Informações Coletadas

O PinFlow coleta apenas as informações necessárias para seu funcionamento:

  • Credenciais de API: App ID, App Secret e Access Token do Pinterest, armazenados localmente no seu navegador (localStorage). Esses dados nunca são enviados a servidores externos além da própria API do Pinterest.
  • Conteúdo de Pins: títulos, descrições, imagens e links que você insere para publicação. Esses dados são processados localmente e enviados diretamente à API do Pinterest.
  • Configurações de uso: preferências de horário, fuso horário e fontes RSS, armazenadas localmente no seu navegador.

💾 2. Onde os Dados são Armazenados

Todos os dados do PinFlow são armazenados exclusivamente no seu dispositivo, via localStorage do navegador. O PinFlow não possui banco de dados próprio nem servidor de backend.

✅ Sem servidor próprio
O PinFlow é um app 100% client-side (roda no seu navegador). Seus dados não passam por nenhum servidor intermediário além da API oficial do Pinterest.

🔗 3. Compartilhamento de Dados

O PinFlow não compartilha, vende ou transfere seus dados pessoais para terceiros. As únicas transmissões de dados realizadas são:

  • Envio de Pins diretamente para a API oficial do Pinterest, conforme ação do usuário
  • Geração de texto via API da Anthropic (Claude) quando o Gerador IA é utilizado — apenas o prompt descrito pelo usuário é enviado, sem dados pessoais

🤖 4. Uso de Inteligência Artificial

O recurso "Gerador IA" utiliza a API da Anthropic para criar sugestões de título, descrição e hashtags. Ao usar esta funcionalidade:

  • Apenas o texto do prompt (tema do Pin) é enviado à Anthropic
  • Nenhuma imagem, credencial ou dado pessoal é transmitido
  • A Anthropic possui sua própria política de privacidade

🔑 5. Segurança das Credenciais

As credenciais da API do Pinterest (App Secret e Access Token) são sensíveis. Recomendamos:

  • Não compartilhar o acesso ao dispositivo onde o PinFlow está configurado
  • Revogar o Access Token no painel do Pinterest caso suspeite de uso indevido
  • Usar o app somente em dispositivos pessoais e de confiança

🧹 6. Exclusão de Dados

Para remover todos os dados armazenados pelo PinFlow do seu dispositivo, basta limpar o localStorage do navegador:

  • No Chrome: Configurações → Privacidade → Limpar dados de navegação → Dados de sites
  • No Firefox: Configurações → Privacidade → Limpar dados → Dados de sites
  • Ou acesse o Console do navegador (F12) e execute: localStorage.clear()

📅 7. Alterações nesta Política

Esta política pode ser atualizada para refletir mudanças no funcionamento do app ou em requisitos legais. A data de atualização sempre será indicada no topo da página. O uso continuado do PinFlow após alterações implica concordância com a versão vigente.
