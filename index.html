📖 Guia de instalação

Para usar o PinFlow com a Pinterest API real, siga os passos:

1. Acesse developers.pinterest.com e crie uma conta de desenvolvedor.

2. Crie um novo app e defina a URL de redirecionamento OAuth como http://localhost:3000/callback

3. Copie o App ID e App Secret para as credenciais acima.

4. Para produção, hospede este app em um servidor e ajuste o redirect URI.

5. A Pinterest API v5 permite criar Pins, gerenciar Boards e agendar conteúdo.
